Emgenisys — embryo classification

A deep-learning system for bovine embryo analysis from short time-lapse clips, built to support grade, stage, sex, and pregnancy prediction with readable model outputs.

View the system

Domain

Embryo imaging

Model

CNN + temporal ML

Input

30-second clips

Output

Grade + prediction

Project details

Emgenisys analyzes bovine embryo time-lapse clips and turns visual development signals into structured predictions. The work centered on a clean video pipeline, reliable model outputs, and evaluation that clinicians and product teams could understand.

Problem

Embryo assessment from video is subtle, time-dependent, and difficult to standardize when experts review clips manually.

Build

Prepared video clips, extracted model-ready frames and temporal signals, and trained modules for embryo state prediction.

Outputs

Produced predictions for grade, stage, sex, and pregnancy likelihood with summaries that support review and iteration.

Stack

PyTorch CNN time-lapse classification evaluation
